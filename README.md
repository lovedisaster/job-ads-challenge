# job-ads-challenge

## How to run the project
1. Environment / Global installs : Node 10, Webpack, Jest 
2. Install : "npm install / yarn install" 
3. Run build : "npm start" 
4. Run server : "npm run server" 
5. Run test : "npm run test"
